What's a "cube"? A custom pool of Magic cards you draft from instead of sealed boosters.

What's a "Scryfall search"? A way to describe a set of cards in words. You type it into the query box. For example:

  • set:vow — every card in the set Crimson Vow
  • t:dragon — every Dragon
  • c:r t:instant — red instants
  • cube:vintage — Wizards' official Vintage Cube list

Not sure what to type? Click an example button under any query box. Full syntax lives on Scryfall.

What's "as-fan"? The average number of a given kind of card you open per pack — so you can see your draft isn't, say, flooded with lands or starved of removal.
