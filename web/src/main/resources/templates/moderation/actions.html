Run Discord moderation actions without leaving the dashboard. You need the matching Discord permission (Ban / Kick / Moderate / Manage) on the bot AND on yourself for each action.

Kick member

Remove a member from the server. They can rejoin via an invite link. Requires the Kick Members permission.

Ban member

Remove a member and prevent them rejoining until they're unbanned. Optionally delete their messages from the past N days (0-7). Requires the Ban Members permission.

Unban user by ID

Lift a ban so a previously-banned user can rejoin. Discord user IDs are 17-19 digit numbers; right-click a user in Discord with Developer Mode on to copy theirs.

Timeout member

Mute a member across every channel for a set duration — they can still see messages but can't send, react, or speak in voice. Discord caps timeouts at 28 days (40320 minutes). Requires the Moderate Members permission.

Remove timeout

Clear an active timeout early so the member can talk again.

Purge messages

Bulk-delete the last N messages in a channel (max 100), optionally filtered to a single user. Discord only allows bulk-deleting messages from the last 14 days — older ones are skipped automatically. Requires Manage Messages.

Lock / unlock channel

Toggle whether @everyone can send messages in a text channel. "Lock" denies Send Messages on @everyone's permission override; "Unlock" clears the override so the role default applies again. Requires Manage Permissions.

Slowmode

Force a minimum delay between each user's messages in a channel. Set to 0 to disable. Max 21600 seconds (6 hours). Requires Manage Channel.
